LinkList类

  1. LinkList类定义

    双链表实现的list和Deque接口。实现所有可选的列表操作，并允许所有元素(包括null)。
    对双链表执行的所有操作都与预期的一样。索引到列表中的操作将从列表的开始或结束(以更接近指定索引的为准)遍历该列表。该类不是线程安全的,并发情况下通过Collections.synchronizedList()包装 Deque里面定义了列表的一些相关操作

      E unlink(Node<E> x) {
    // assert x != null;
    final E element = x.item;
    final Node<E> next = x.next;
    final Node<E> prev = x.prev;
    //如果前结点为null，则头结点指向当前结点的下一个结点
    if (prev == null) {
        first = next;
    } else { //否则前结点的下一个结点指针指向当前结点的下一个结点，并把当前结点的前结点指针置空
        prev.next = next;
        x.prev = null;
    }
    //当前结点的下一个结点处理类似
    if (next == null) {
        last = prev;
    } else {
        next.prev = prev;
        x.next = null;
    }
    //清空元素
    x.item = null;
    size--;
    modCount++;
    return element;
}
