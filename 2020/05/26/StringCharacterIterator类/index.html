StringCharacterIterator类

  1. StringCharacterIterator类定义

    StringCharacterIterator实现了字符串的CharacterIterator接口。StringCharacterIterator类遍历整个字符串。
    类中的几个属性

  2. 构造方法

    1. StringCharacterIterator(String text)

      传入要遍历的字符串，遍历的游标为下标0开始

    2. StringCharacterIterator(String text, int pos)

      遍历字符串从指定游标位置开始

    3. StringCharacterIterator(String text, int begin, int end, int pos)

      遍历字符串从指定游标位置开始，遍历范围在begin-end之间

  3. 其他方法

    1. setText(String text)

      传入要迭代的字符串，重置该迭代器

    2. current()

      返回当前位置的字符

    3. last()

      返回最后一个位置的字符

    4. setIndex(int p)

      返回指定下标处的字符

    5. next()

      获取下一个位置的字符，如果当前位置大于末尾下标则表示遍历已经结束的标识

    6. previous()

      获取前一位置的字符
