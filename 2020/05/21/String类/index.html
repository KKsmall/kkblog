String类

  1. String类定义

    String类表示字符串。Java程序中的所有字符串文本(如“abc”)都实现为该类的实例。字符串都是常量;它们的值在创建之后不能更改。字符串缓冲区支持可变字符串。因为字符串对象是不可变的，所以它们可以被共享，CharSequence定义了一些字符串相关操作的重要接口

  2. String的几个属性

  3. 构造方法

    String构造方法有用多种，我们依次看看

    1. String()

      初始化新创建的字符串对象，使其表示空字符序列。注意，使用这个构造函数是不必要的，因为字符串是不可变的

    2. String(String original)

      初始化新创建的字符串对象，使其表示与参数相同的字符序列;换句话说，新创建的字符串是参数字符串的副本。
      除非需要显式的original副本，否则没有必要使用这个构造函数，因为字符串是不可变的。注意这里传入的是String对象

    3. String(char value[])

      分配一个新字符串，使它表示字符数组参数中当前包含的字符序列。复制字符数组的内容;字符数组的后续修改不影响新创建的字符串。
      跟上边的区别是这里只是对参数值进行复制

    4. String(char value[], int offset, int count)

      依据value[],获取偏移量offset下标开始的count个元素作为子数组创建String对象，新创建的对象不收字符数组的影响

    5. String(int[] codePoints, int offset, int count)

      这里的int数组存放的是Unicode的code point值，同样截取一定长度创建String对象

    6. String(byte bytes[], int offset, int length, String charsetName)弃用

      分配一个由8位整数值数组的子数组构造的新字符串。
      偏移量参数是子数组的第一个字节的索引，而计数参数指定子数组的长度。
      子数组中的每个字节都被转换成一个char，方法被弃用了。

    7. String(byte bytes[], int offset, int length, Charset charset)弃用

    8. String(byte bytes[], String charsetName)

      通过使用指定的字符集解码指定的字节子数组构造新的字符串。

      新字符串的长度是字符集的函数，因此可能不等于子数组的长度，我将中文“我们好啊发送”字符串转成byte数组调用该方法测试。

    9. String(byte bytes[])

      内部还是调用上面已经定义好的方法

    10. String(StringBuffer buffer)

      分配一个新字符串，该字符串包含字符串缓冲区参数中当前包含的字符序列。复制字符串缓冲区的内容,字符串缓冲区的后续修改不影响新创建的字符串

    11. String(StringBuilder builder)

  4. 其他常见方法

    1. length方法

      直接获取字符数组的长度

    2. isEmpty方法

      判断字符数组长度是否为0

    3. charAt方法

      获取下标处的字符

    4. codePointAt方法

      获取指定下标处的字符的unicode编码

    5. codePointBefore方法

      获取下标前一位的元素的unicode编码

    6. codePointCount方法

      计算字符串范围内unicode字符的个数

    7. offsetByCodePoints方法

      返回该字符串中被codePointOffset代码点从给定索引(index)偏移(codePointOffset)后的的索引。

    8. getChars方法

      将字符串相应范围内的字符复制起始位置为dstBegin的dist数组里

    9. getBytes方法

      获取字符串的byte数组

    10. equals方法

      重写了Object的equals方法，修改为比较字符串内容

    11. contentEquals方法

      CharSequence是上层接口，String,StringBuffer,StringBuilder等都实现了该接口

    12. compareTo方法

      先比较两个字符数组里的字符，比较范围是最短数组的长度，如果都相等则比较二者长度

    13. compareToIgnoreCase方法

      调用私有内部类的方法去比较，逻辑大致同上，只是在循环中增加了把两个比较字符转成大写比较和转成小写比较

    14. startsWith方法

      判断字符串从指定偏移下标开始是否以指定字符串开头
      startsWith(String prefix)里面默认参数为0startsWith(prefix, 0)

    15. endsWith方法

      endsWith方法内部也是调用的startsWith方法，只比较末尾suffix长度的字符

    16. indexOf方法

    17. substring方法

      //截取下标之后的字符数组，创建新的字符串对象

    18. concat方法

      合并两个字符串

    19. replace方法

      字符串中字符替换

      该方法接收参数为CharSequence接口，判断当前字符串时候有目标字符串，把符合的部分替换

    20. split方法

    21. join方法

      静态方法，根据分割符delimiter拼接字符串

    22. trim方法

      清楚字符串前后的空格符

      public String trim() {
    int len = value.length;
    int st = 0;
    char[] val = value;    /* avoid getfield opcode */

    while ((st < len) && (val[st] <= ' ')) {
        st++;
    }
    while ((st < len) && (val[len - 1] <= ' ')) {
        len--;
    }
    return ((st > 0) || (len < value.length)) ? substring(st, len) : this;
}
